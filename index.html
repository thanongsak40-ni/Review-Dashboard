API Documentation

Back Office API (POST)

  • • getProjectsByStatus - { status: 'Skip'|'Success' }
  • • getPendingReviews - Get pending review list
  • • getCompletedReviews - Get completed review list
  • • updateReview - { rowIndex, jobNumber, project, houseNumber }
  • • updateReviewStatus - { rowIndex, status, imageUrl }
  • • addNote - { rowIndex, note }

URL: http://localhost:3000

Dashboard API (GET)

  • • getDashboardData
  • Parameters:
    • - period: 'this-month'|'this-year'|'by-month'|'by-year'
    • - month: 1-12 (if by-month)
    • - year: YYYY
    • - projects: comma-separated list

URL: http://localhost:3000?action=getDashboardData&...

💡 Note: All data is stored in memory. Changes will be reset when the server restarts.
